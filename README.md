## 1. 我们收集的信息

我们或我们的第三方合作伙伴提供服务时，可能会收集、储存和使用下列与您有关的信息。如果您不提供相关信息，可能无法注册成为我们的用户或无法享受我们提供的某些服务。

- **账户信息**：手机号码（用于登录验证）

### 1.2 系统自动收集的信息

- **设备信息**：设备型号、操作系统版本（用于应用优化和兼容性）
- **使用数据**：应用使用频率、功能使用情况、交互行为（用于改进服务）
- **日志信息**：您的操作记录、时间戳等（用于安全防护）

为了向您提供更好的服务，我们接入以下第三方服务：

## 2. 信息的存储

### 2.1 存储方式和期限

我们会通过安全的方式存储您的信息，包括：
- 本地存储（应用进行数据缓存）
- 加密的云端服务器
- 数据库和服务器日志

一般情况下，我们只会在为实现服务目的所必需的时间内或法律法规规定的条件下存储您的个人信息。

### 2.2 存储地域

我们会按照法律法规规定，将境内收集的用户个人信息存储于中国境内。

目前我们不会跨境传输或存储您的个人信息。将来如需跨境传输或存储的，我们会向您告知信息出境的目的、接收方、安全保证措施和安全风险，并征得您的同意。

### 2.3 产品停止运营时的处理

当我们的产品或服务停止运营时，我们将以推送通知、公告等形式通知您，并在合理期限内删除您的个人信息或进行匿名化处理（法律法规另有规定的除外）。

## 3. 信息安全

我们采取多层次的安全措施保护您的信息：

- **加密传输**：使用TLS加密技术保护数据传输
- **加密存储**：对话内容和个人数据使用加密方式存储
- **访问控制**：实施严格的权限管理，仅授权人员可访问用户数据
- **脱敏处理**：对话内容经过脱敏处理，不会与您的身份信息直接关联
- **定期审计**：定期进行安全审计，确保数据安全
- **用户密码**：密码使用bcrypt算法加密

但请您理解，由于技术的限制以及可能存在的各种恶意手段，在互联网行业，即便竭尽所能加强安全措施，也不可能始终保证信息百分之百的安全。

我们可能将在向您提供服务的过程中所收集的信息用作下列用途：

- **提供核心服务**：深度探索对话、复盘总结、模式分析
- **身份验证和安全**：账户验证、防止恶意使用、诈骗监测、安全防范
- **个性化体验**：基于您的对话历史和使用习惯生成个人模式洞察
- **服务改进**：分析用户行为趋势、优化应用功能和用户体验
- **客户服务**：及时处理您的问题和反馈
- **存档和备份**：确保数据的可恢复性和服务连续性

**我们不会将您的数据用于广告投放或其他商业目的。**

## 6. 信息共享

### 6.1 共享原则

目前，我们不会主动共享或转让您的个人信息至第三方。如存在其他共享或转让您的个人信息的情形，我们会直接向您说明，并征得您的明示同意。

### 6.2 第三方合作

我们与第三方合作伙伴（如Apple、阿里云、DeepSeek、腾讯云）合作以提供服务。这些合作伙伴已承诺遵守我们关于数据隐私保护的措施与要求，包括但不限于数据保护协议和相关处理政策。

### 6.3 个人身份识别信息保护

我们不会向合作伙伴分享可用于识别您个人身份的信息（例如您的姓名或电子邮件地址），除非您明确授权。

### 6.4 法律要求下的信息共享

根据相关法律法规及国家标准，以下情形中，我们可能会共享、转让、公开披露个人信息，无需事先征得您的授权同意：

- 与国家安全、国防安全直接相关的
- 与公共安全、公共卫生、重大公共利益直接相关的
- 犯罪侦查、起诉、审判和判决执行等直接相关的
- 出于维护个人信息主体或其他个人的生命、财产等重大合法权益但又很难得到本人同意的
- 个人信息主体自行向社会公众公开个人信息的
- 从合法公开披露的信息中收集个人信息的

### 6.5 业务变更

随着我们业务的持续发展，我们有可能进行合并、收购、资产转让等交易。在此情况下，我们将告知您相关情形，按照法律法规及不低于本《隐私政策》所要求的标准继续保护或要求新的控制者继续保护您的个人信息。

您有权对您的个人信息进行以下操作：

- **查询权**：查询和了解我们持有的您的个人信息
- **更正权**：更正不准确或不完整的个人信息
- **删除权**：删除您的账户和相关数据
- **撤回同意权**：撤回对数据处理的同意

### 申请方式

您可以通过以下方式行使上述权利：
1. 直接在应用内的"设置"页面进行相关操作
2. 通过应用内的"意见提交"功能提出请求
3. 发送邮件至 xinyuhoujerry@gmail.com

我们会按照法律法规的规定在合理期限内作出反馈。

### 账户注销

当您决定不再使用我们的产品或服务时，可以申请注销账户。注销后，除法律法规另有规定外，我们将在7天内删除或匿名化处理您的个人信息。您也可以在注销前7天内重新登录以恢复账户。
